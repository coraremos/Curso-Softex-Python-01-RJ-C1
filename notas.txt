configurar o django para templates: (VER APOSTILA 2)

    1. criar a pasta templates na raiz do projeto
    2. editar a linha TEMPLATES -> 'DIRS' (de diretórios) do settings (do projeto) colocando: 
        'DIRS': [BASE_DIR/'templates'],
    3. criar o arquivo .html
    4. renderizar no views do app -> criar no view do core uma def com o nome da página (ex.: home) pra retornar uma render
    5. incluir a configuração no urls do app -> criar arquivo urls.py com um mapa
    6. incluir o mapa do app no urls do projeto -> chamar o urls do app no urls do projeto (importanto include)

    7. passar context para o template (editar o views do app e depois o html)
        
        O Django usa uma sintaxe especial chamada Django Template Language (DTL). 
        ●  {{ variavel }}: Usado para exibir uma variável. 
        ●  {% tag %}: Usado para executar lógica (como if ou for).
        ●  {% block ... %} define "buracos" que as páginas "filhas" podem preencher. 

    8. adicionar CSS

        criar a pasta static
        adicionar/colar abaixo da linha STATIC_URL = 'static/' a informação:
            STATICFILES_DIRS = [BASE_DIR / 'static']
        dentro da pasta 'static' criar pasta 'css' e, dentro, o arquivo 'style.css'
        adicionar a estilização q vc quiser nele
        no arquivo 'base.html' colocar no topo:
            {% load static %}
        e no <head> após o tílulo, colocar:
            <link rel="stylesheet" href="{% static 'css/style.css' %}">
        no topo de 'home.html' colocar:
            {% extends 'base.html' %}
